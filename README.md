# code-skill-validator-python
Your directory should look like this:
```
factorial
├── factorial.py
├── README
└── setup.py
```
You want to create setup.py in your repo like so.

```python
from setuptools import setup

setup(
   name='factorial',
   version='1.0',
   description='',
   author='',
   author_email='',
   install_requires=[], #external packages as dependencies
)
```
